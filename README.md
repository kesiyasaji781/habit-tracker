# Smart Habit Tracker

A full-stack habit tracking web application built using Flask and SQLite.

## Features

- User Registration & Login
- Habit Management
- Habit Streak Tracking
- Weekly Analytics
- Progress Dashboard
- Dark Mode
- Responsive Design

## Tech Stack

- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript
- Bootstrap
- Chart.js

## Installation

pip install -r requirements.txt

python app.py

Open:
http://127.0.0.1:5000
